\doxysubsection{Detailed Description}
Maintains a registry of network-\/connected modules and returns \doxylink{class_pnd_1_1_group}{Group} objects to the user. 

Only one \doxylink{class_pnd_1_1_lookup}{Lookup} object is needed per application.
